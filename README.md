# Azure IAM Security Labs

This repository contains hands-on Azure Identity and Access Management (IAM) security labs focused on designing and implementing enterprise-grade identity controls using Microsoft Entra ID.

## Focus Areas
- Azure Entra ID (Azure AD)
- Identity and Access Management (IAM)
- Role-Based Access Control (RBAC)
- Least Privilege Access
- Zero Trust Identity Principles
